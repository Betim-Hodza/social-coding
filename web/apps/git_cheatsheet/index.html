Git cheatsheet

Regularly used commands for day to day operations

Commands

Command Description
git clone [url] Clone a repository
git status Show modified files in directory, staged for commit
git add [path of file] Stages file created/updated
git commit -m "message" Commits the staged file to repository with a message
git push [alias] [branch] Push local changes in the machine to the repository on the internet
git stash Save modified and staged changes


Conventional commits

Commit type Defintion Example
feat Highlights that a feature has been added to an implementation git commit -m "feat: added square root (sqrt) operation to calculator app"
fix Highlights that a previous feature that had a bug has been fixed git commit -m "fix: equal button in calculator app"
docs Presents change in documentation in terms of comments to code, doc strings or changes in readme git commit -m "docs: Updated parameters used in add function"
